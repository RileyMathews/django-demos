Welcome to Game stone!

This is my frontend capstone project from my time at Nashville Software School.

It was originally written in pure react with a simple dummy backend for demo purposes. I have since re-written it with a full Django REST framework backend and react components served up from Django templates.

Game data is sourced by the Giantbomb API. Go check them out! It's a cool site!
